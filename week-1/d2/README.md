# Day 2 – Python Fundamentals

## Objective

The goal of Day 2 was to practice the main Python concepts needed before
working with NumPy, Pandas, and machine learning libraries.

## Topics Covered

During this day, I studied and practiced:

- Variables and basic data types
- Lists, tuples, dictionaries, and sets
- Arithmetic, comparison, logical, and membership operators
- Conditional statements using `if`, `elif`, and `else`
- `for` loops and `while` loops
- `range()`, `break`, `continue`, and `enumerate()`
- Functions, parameters, arguments, and return values
- Default parameters and docstrings
- Mutable and immutable function arguments
- List comprehensions
- Basic object-oriented programming
- Classes, objects, attributes, methods, `__init__`, and `self`

## Exercises Completed

The notebook includes the required Day 2 exercises:

1. A function that receives a list of numbers and returns the mean,
   minimum, and maximum in a dictionary.
2. Filtering even numbers using a normal `for` loop.
3. Filtering the same values using a list comprehension.
4. A small `StudentRecord` class containing attributes and methods.
5. Markdown explanations describing the purpose of the main sections.
