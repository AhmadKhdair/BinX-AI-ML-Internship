## Integrated Mini-Notebook

The final Week 1 task is:

`week_1_integrated_mini_notebook.ipynb`

The project uses the Titanic dataset and follows a complete basic data analysis workflow:

1. Loading and inspecting the dataset
2. Checking data quality
3. Cleaning missing values and duplicate rows
4. Creating derived features with NumPy
5. Filtering and grouping the data
6. Creating labeled visualizations
7. Writing findings and conclusions

The project includes:

- Histogram of passenger ages
- Scatter plot of passenger age and ticket fare
- Bar plot of survival rate by passenger class
- Subplots comparing survival by sex and travelling status
